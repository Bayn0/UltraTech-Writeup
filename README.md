## Description

This repository contains a complete write-up and supporting artifacts for the **UltraTech** room from the TryHackMe platform.

The objective of this challenge is to demonstrate a structured attack methodology, including:
- Network and web enumeration
- Identification and exploitation of a backend vulnerability
- Privilege escalation through a system misconfiguration

Both **French** and **English** versions of the write-up are provided.
